# Online Pizza Store

## Project Overview

Online Pizza Store is a web-based application that allows customers to browse pizzas, place orders, and manage their purchases through an easy-to-use interface. The system is designed to simplify online food ordering and provide a smooth user experience.

## Features

* User Registration and Login
* Browse Pizza Menu
* Add Pizzas to Cart
* Place Orders Online
* View Order Details
* Manage Customer Information
* Responsive and User-Friendly Interface

## Technologies Used

* HTML
* CSS
* JavaScript
* JSP / Servlets (if using Tomcat)
* MySQL Database
* Apache Tomcat Server
* JDBC

## Project Structure

```text
OnlinePizzaStore/
├── src/
├── WebContent/
├── WEB-INF/
├── css/
├── js/
├── images/
└── database/
```

## Prerequisites

Before running the project, ensure that you have:

* Java JDK 8 or above
* Apache Tomcat Server
* MySQL Database
* Eclipse IDE / NetBeans (optional)

## Database Setup

1. Create a MySQL database.
2. Import the provided SQL file.
3. Update database connection details in the project.
4. Start MySQL Server.

## How to Run

1. Import the project into Eclipse or your preferred IDE.
2. Configure Apache Tomcat Server.
3. Add the project to the server.
4. Start the server.
5. Open a browser and access:

```text
http://localhost:8080/online pizza store
```
